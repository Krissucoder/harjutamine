# GitHubi tutvustus

GitHub on veebipõhine versioonihaldusplatvorm, mis kasutab Git-i.
Seda kasutatakse koodi, dokumentide ja projektide haldamiseks ning
meeskonnatöö lihtsustamiseks.
## Minu mõtted GitHubi kohta

GitHub on kasulik tööriist, sest see võimaldab hoida projektide ajalugu,
teha koostööd teistega ja katsetada uusi ideid ilma põhiversiooni rikkumata.
Kavatsen GitHubi edaspidi kasutada nii õppetöös kui ka isiklikes projektides.
